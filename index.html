Minima.l

Opinionated LISP dialect that takes some of its inspirations from Picolisp, CHICKEN Scheme, and other great languages. The source code is located here.

(load '(math + - <=) '(io prinl) '(std if))

(def fib (N)
  (if (<= N 1)
    N
    (+ (fib (- N 1)) (fib (- N 2)))))

(prinl "Result: " (fib 30))
